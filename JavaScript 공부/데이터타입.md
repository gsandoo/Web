# 데이터 타입

- 문자열(text) , 숫자(number)

# 문자열

- 따음표를 입력하고 문자열을 입력. 

- 예: const value="hello javascript !";

# 숫자

- 따음표 없이 그냥 입력

- 예: let number=123;

# 불리언 boolean (참 혹은 거짓)

- 참과 거짓.

- 예: let number= true;
