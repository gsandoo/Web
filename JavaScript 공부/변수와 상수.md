# 변수

- 변하는 값.

- <strong>let</strong>을 통해 변수를 선언한다

- let 을 통한 변수는 다시 똑같은 이름으로 선언할 수 없다.

- 예: let value=1;
      console.log(value);
      = 1

-     let value=1;
      value=2;
      console.log(value);
      =2

# 상수

- 변하지 않는 값. 고정적인 값 . 한번 설정하면 바뀌지 않음.

- <strong>const</strong>를 활용하여 상수를 나타냄.

- 예: const value=1;
      console.log(value);
      = 1

      const value= "hello";
      const value= "hi";  >>> 잘못됌
