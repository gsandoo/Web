# if 문
- if 와 else 로 가능.

- if 다음에 조건을 () 로 묶고 콜론 없이 중괄호로 이어붙힌다.

- else는 바로 이어 붙히기.

- 여러 조건일시 elif 대신 else if 를 입력.


- <pre> const a = 1;
        if (a === 1){
            console.log("a는 1 입니다.")
        }else if (a===2){
            console.log("a는 2 입니다.")
        }else{
            console.log("a는 1이 아닙니다.")
        }</pre>
