# IFRAME 태그

- 다른 html 페이지를 현재 페이지에 삽입.

<strong>속성:</strong>

- name
프레임 이름

- src
URL

- width
가로넓이

- height
세로넓이

- allowfullscreen
전체화면사용여부

- frameborder (기본적으로 1)
프레임 테두리 사용 여부

-sandbox
보안을 위한 읽기 전용으로 삽입

<strong>속성:</strong>

- width

- height
