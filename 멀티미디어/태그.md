# src 속성 

- 이미지를 2장이상 준비
- 이미지 크기가 px 이 아닌 w 단위 혹은 x를 사용.
- 작은 이미지 부터 순서대로 입력.
- w는 가로너비
<strong>예)
srcset="이미지 경로 (한칸뛰고) (원본크기)w ,"</strong>
